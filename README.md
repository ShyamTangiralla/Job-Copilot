# Job Copilot – Job Application Tracking & Analytics Platform

## Overview
Job Copilot is a full-stack job application management and analytics platform designed to help job seekers track applications, manage resume versions, monitor interview pipelines, and analyze job search performance.

## Why I Built This
I built this project to solve the problem of managing hundreds of job applications across multiple platforms. Instead of using spreadsheets, I created a system that tracks applications, resumes, networking contacts, and analytics in one place.

## Key Features
- Job application tracking dashboard
- Resume version management
- Interview and offer tracker
- Networking contact tracker
- Job analytics and performance metrics
- Resume export to PDF and DOCX
- ATS keyword optimization and scoring
- Job pipeline management
- Job source tracking and analytics

## Analytics Features
- Applications per week/month
- Interview conversion rate
- Offer rate tracking
- Average days to interview
- Recruiter response tracking
- Job source performance analysis

## Tech Stack
- Frontend: React / TypeScript
- Backend: Node.js / Express
- Database: PostgreSQL
- ORM: Drizzle
- Styling: Tailwind CSS
- Hosting/Dev: Replit
- Version Control: GitHub

## Project Modules
- Dashboard / Analytics
- Job Discovery & Import
- Jobs Inbox / Pipeline
- Resume Vault / Resume Versions
- Interview Tracker
- Offer Tracker
- Networking Tracker
- Candidate Profile
- Export Center

## Future Improvements
- Chrome extension for saving jobs
- Email integration for job alerts
- Job market analytics dashboard
- Resume scoring AI
- Interview preparation tracker
- Networking follow-up reminders
- Automated job application tracking

## How to Run Locally
```bash
git clone https://github.com/ShyamTangiralla/Job-Copilot.git
cd Job-Copilot
npm install
npm run dev
```
